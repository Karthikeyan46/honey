# SilvanNectar™ - Premium Forest Honey

Direct from the heart of the forest to your table. Experience the raw, unfiltered essence of nature in every drop.

## Features
- **Direct Sourcing**: Partnering with traditional forest harvesters.
- **Pure & Certified**: Lab-tested, 100% Organic, and Forest Born.
- **Rare Collection**: Forest Wild, Kombu, and Murangai Honey varieties.
- **Premium Design**: Responsive, dark-mode experience with smooth scroll animations.

## How to View
This is a static web application. Simply open `index.html` in any modern web browser.

## Deployment
Hosted on Vercel via GitHub integration.
